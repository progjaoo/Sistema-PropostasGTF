# Documentacao do Sistema de Propostas

Este diretorio centraliza as guidelines, padroes tecnicos e regras funcionais do Sistema de Propostas.

## Identificacao do Projeto

- Nome: Sistema de Propostas
- Tipo: WEB + API
- Stack principal: TypeScript, React, Vite, Express, Prisma, PostgreSQL, Docker e PNPM Workspaces.
- Objetivo: gerenciar empresas, programas, produtos, anunciantes e propostas comerciais com perfis ADMIN e COMERCIAL.
